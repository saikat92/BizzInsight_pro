# Business Intelligence System

A comprehensive desktop application for business analytics, data management, and machine learning predictions.

## Features
- Database management with SQLite
- Interactive Tkinter dashboard
- Sales trend analysis and visualization
- Customer segmentation
- Product performance tracking
- Machine learning for sales prediction
- Report generation (PDF, Excel, CSV)
- Data entry forms
